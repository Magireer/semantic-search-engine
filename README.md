# Semantic Search Engine

![FAISS](https://img.shields.io/badge/FAISS-v1.7.4-green) ![Transformers](https://img.shields.io/badge/Transformers-v4.30-blue)

Efficiently search through millions of documents using dense vector embeddings and approximate nearest neighbor search.

## Features
- Fast indexing with FAISS
- Support for multiple embedding models
- REST API for querying
